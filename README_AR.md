# ALRAKEEN | Teltonika CAN Web Pro

نسخة ويب قوية (Next.js) تعمل على Vercel بدون أي إعدادات إضافية.

## تشغيل محلي (Windows)
1) افتح CMD داخل فولدر المشروع
2) شغّل:
   - install.cmd (مرة واحدة)
   - run.cmd

## نشر على Vercel
- ارفع المشروع على GitHub
- اربطه بـ Vercel → Deploy

## الاستخدام
- Upload Excel
- اختر خيارات التقرير (خطة تركيب / تكاليف / PDF)
- Download ZIP يحتوي:
  - ALRAKEEN_Output.xlsx
  - ALRAKEEN_Report.pdf (إذا مفعّل)
